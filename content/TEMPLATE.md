## script.md (short-form, 30–45s)

```
HOOK (0–3s, direct-to-camera, verbatim): "..."
THE CHART (3–15s): show visual; state the ONE claim it makes.
SO WHAT (15–35s): consequence for the persona, in their language.
CTA (35–45s): the checkable action on TERREM. "Link in bio →  <page>"
```

Rules: no greeting, no intro, no "in this video." First frame = analyst mid-sentence + chart visible. Captions burned in (36–60px, safe zones).
